# git教程

  • 查看git的版本
[root@VM-0-9-centos software]# Git --version
Git version 2.3.0

  • git add . 添加项目到缓存

  • git commit -m '描述' 提交到本地

  • 推送到远程 git push origin 分支的名字(master)

# 提交到远程

# 提交

  1. 不提交 node_modules

  1. 初始化 git仓库

    Git init

  2. 提交到缓存

    git add 某个文件
git add .

  3. 提交到本地

    Git commit -m 描述
Git commit -m '人生第一次'

  4. 查看是否关联远程地址

    Git remote -v

  5. 关联远程仓库地址远程

    Git remote add origin 远程的地址
Git remote add origin https://gitee.com/bingyu123/server.git

  6. 提交到远程

    Git push origin 分支的名字
Git push origin master

# 拉

  • 在服务器拉取

  • /opt/project中拉取

  • cd /opt/project

  • 克隆 git clone 地址

  • cd /server && npm i 下载依赖

  • node index.js 启动项目

  • 从远程拉取最新文件

    Git pull origin 分支
Git pull origin master

  • 启动服务器 node index.js
