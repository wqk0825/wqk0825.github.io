# Git命令

# 创建新仓库

git init //以创建新的 Git 仓库

# 克隆远端服务器上的仓库

git clone https://github.com/hape666/cdn.Git

镜像克隆 使用国内镜像，目前已知Github国内镜像网站有github.com.cnpmjs.org和git.sdut.me/。速度根据各地情况而定，在clone某个项目的时候将github.com替换为github.com.cnpmjs.org即可。

# 实例

//这是我们要clone的
git clone https://github.com/Hackergeek/architecture-samples
 
//使用镜像
git clone https://github.com.cnpmjs.org/Hackergeek/architecture-samples
 
//或者
 
//使用镜像
git clone https://Git.sdut.me/Hackergeek/architecture-samples

# 添加和提交

# 添加

添加到暂存区，使用如下命令：
git add <filename>
git add *
git add .

# 提交

git commit -m "代码提交信息"

# 提交到远程仓库

git push origin master

# 关联远程仓库

git remote add origin 远程地址

# 显示所有远程仓库

git remote -v


git pull origin master --allow-unrelated-histories

# git : 码云 github gitlab

# 基于个⼈

Git clone 地址 : 克隆⼀个仓库地址

Git init : Git 初始化 ⽣成 .Git⽂件

Git add ⽂件名字 : Git add . : 将所有⽂件提交到缓存 没有提交记录 

Git commit -m '描述信息' : 提交到本地

Git log : 提交⽇志

Git reset --hard hash值 回到此hash值的记录

Git status : 查看git状态

Git remote add origin 地址 : 关联远程地址

Git remote -v : 查看有没有关联远程地址

Git push origin 分⽀ : 推送到远程

Git pull : 从远程拉取

Git pull origin 分⽀名字

# 团队协作 : 分⽀

Git checkout  创建分支

命令 : 以姓名命名 :

默认分⽀(主分⽀) : master

合并 : dev分⽀

Git checkout -b AA 创建分支 并切换到此分支

Git branch 查看所有分支信息

Git branch AA

Git branch BB

Git checkout AA

Git merge 分⽀名字 : 合并分⽀

Git rm -r --cached .   清除add提交 缓存区的内容
